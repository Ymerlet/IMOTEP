# IMOTEP - Micro-Meteorological Model for Outdoor Thermal Environment Perception

IMOTEP is a micro-meteorological model that simulates the thermal environments of urban scenes 
that may contain built shelters, building overhangs or trees, and evaluates outdoor thermal 
comfort.

## Installation
On a terminal : 
1. clone the repo on your computer : `git clone https://github.com/tgresse/IMOTEP.git`
2. Change current folder `cd ./IMOTEP`
3. Install the lib `pip install -e ./

## Usage

Refer to the *IMOTEP_userguide.docx* user guide and the *IMOTEP_techspecs.docx* technical 
documentation in the *\doc* folder.
